# 🎬 영화 추천 및 탐색 웹 페이지  
## UI/UX 구현 중심 프론트엔드 팀 프로젝트

![Image](https://github.com/user-attachments/assets/ef1f9591-9faf-4a1a-81f0-0bc024d319a4)

이 프로젝트는  
**완성도 있는 화면 구현**을 목표로 진행한  
프론트엔드 퍼블리싱 및 UI/UX 중심 팀 프로젝트입니다.

초기 기획 단계의 구조를 검토한 뒤,  
5일간의 집중 스프린트를 통해 화면 구현과 사용자 인터랙션에 집중해  
웹 페이지를 완성했습니다.

---

## 🛠 Tech Stack
HTML / CSS / JavaScript / Git

---

## 🧩 프로젝트 개요
- **프로젝트 유형**: 프론트엔드 UI/UX 구현 중심 팀 프로젝트 (2인)
- **중점 사항**
  - 퍼블리싱 완성도와 화면 디테일
  - JSON 데이터를 활용한 안정적인 화면 렌더링
  - 사용자 흐름을 고려한 인터랙션 구현

---

## ✨ 주요 구현 내용

### 1. UI/UX 구현
- 메인 페이지 외에도 상세, 탐색, 커뮤니티 등 여러 서브 페이지 구성
- 줄거리 더보기/접기, 플랫폼 아이콘 출력, 밸런스 게임 등 인터랙션 구현
- 다양한 화면 크기를 고려한 반응형 레이아웃 적용

### 2. 데이터 처리 방식
- `<template>` 태그를 활용한 카드 렌더링 구조 설계
- 서버 DB 대신 JSON 데이터를 사용하여 화면을 안정적으로 출력

### 3. 협업 방식
- 2인이 역할을 나누어 5일간 집중적으로 작업
- GitHub PR(Pull Request)을 활용한 병합 및 작업 관리

---

## 🧠 프로젝트를 통해 배운 점
- 공통 UI 요소를 컴포넌트처럼 관리하며 구조를 정리하는 경험
- 접근성을 고려한 마크업(`aria-*`) 적용
- 초기 구조의 한계를 빠르게 판단하고,  
  팀 상황에 맞는 방향으로 작업 범위를 조정하는 경험
